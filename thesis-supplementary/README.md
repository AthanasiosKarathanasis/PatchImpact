# Thesis Supplementary Material

This folder contains supplementary material for the MSc thesis:

**Measuring Software Patch Impact in Open Source Projects: A Heuristic-Based Evaluation Framework**

Author: Athanasios Karathanasis  
Programme: MSc in Software Engineering, University of Southern Denmark  
Submission date of thesis document: 1 June 2026

## Purpose

The files in this folder provide supporting experimental artefacts for the PatchImpact thesis work. They include generated reports, CSV outputs, raw framework outputs, dataset notes, and related reproducibility material.

The submitted thesis document contains representative summaries and selected outputs, while this repository provides additional supporting material for transparency, inspection, and reproducibility.

## Contents

- `reports/` contains generated report files.
- `csv-results/` contains CSV output files produced by the framework.
- `raw-outputs/` contains raw terminal or framework output used during evaluation.
- `dataset-notes/` contains notes about datasets and source repositories used or considered.

## Note

These files are supplementary artefacts connected to the submitted thesis and the PatchImpact framework. They do not replace or modify the submitted thesis document.
